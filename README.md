# cpp-c-map-wrapper
Call a C++ map&lt;string, string&gt; from C code

Install
-------

Install the compiler and build utils:

```
sudo apt install gcc g++ make
```

Build
-----

Run the `make` command:

```
make
```

Run
---

Load the environ variables:

```
source setenv.sh
```

Run the example program:

```
./dist/map_str_str_test
```
